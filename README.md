# IBM Data Science Professional Certificate - Capstone Project

This repository houses all the Jupyter Notebooks (`.ipynb`) developed during the final Capstone Project of the IBM Data Science Professional Certificate. 

### 📌 Project Overview
The project focuses on predicting the success of SpaceX Falcon 9 first-stage landings by leveraging data collection (APIs & Web Scraping), EDA (SQL & Pandas), interactive data visualizations (Folium & Dash), and Machine Learning classification models.

### 🛠️ Tech Stack & Skills Covered
* **Languages:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn, Folium, Plotly Dash
* **Key Concepts:** Data Wrangling, Exploratory Data Analysis (EDA), Predictive Modeling (Logistic Regression, SVM, Decision Trees, KNN), Hyperparameter Tuning (`GridSearchCV`).
